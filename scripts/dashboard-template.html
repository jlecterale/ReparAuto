Automatização do Dashboard

Este dashboard é um snapshot técnico do projeto (git, ficheiros, PRs, roadmap), gerado localmente sob demanda. As métricas operacionais ao vivo (carros, peças, oficinas, intenções, utilizadores, moderação) vivem agora no Painel Admin → Visão Geral do site.

Gerar localmente

Corra o comando abaixo na raiz do projeto sempre que quiser atualizar este ficheiro. O resultado (docs/dashboard.html) é ignorado pelo Git — é um artefacto local, não versionado. 

npm run generate:dashboard

Opcional: regenerar a cada commit (Git Hook local)

Se quiser regenerar automaticamente em cada git commit, crie .git/hooks/post-commit (e chmod +x em Linux/macOS): 

#!/bin/sh
echo "Atualizando o Dashboard do Projeto..."
npm run generate:dashboard
